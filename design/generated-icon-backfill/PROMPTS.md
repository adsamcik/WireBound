# Generated Icon Backfill Prompts

These icons backfill referenced `WbIcon` keys that did not have matching PNG assets. Each was generated as a separate standalone icon on a flat `#00ff00` chroma-key background, then converted to transparent RGBA PNG.

Shared style:

- WireBound in-page icon system, not sidebar nav.
- Aqua/cyan generated glyphs using `#74F5F2` with pale cyan highlight.
- Simple geometric silhouettes with rounded caps/joins and thick strokes for 14-24 px readability.
- No text, labels, logos, shadows, gradients, panels, badges, borders, or 3D.
- Centered square canvas with generous padding.

Generated keys:

- `WbStatusInfo`: bold circled information glyph for notice/status banners.
- `WbEntityApp`: rounded application window tile for app rows and app metrics.
- `WbActionStop`: rounded square stop control for helper stop actions.
- `WbAdapterGeneric`: central connector ring with short horizontal terminals for unknown adapters.
- `WbAdapterWifi1`: one-arc weak Wi-Fi adapter variant matching `wb-adapter-wifi.png`.
- `WbAdapterWifi2`: two-arc Wi-Fi adapter variant matching `wb-adapter-wifi.png`.
- `WbAdapterWifi3`: three-arc Wi-Fi adapter variant matching `wb-adapter-wifi.png`.
- `WbAdapterUsb`: compact USB plug shape with cable stem.
- `WbAdapterBtTether`: two linked device nodes with a small wireless arc.
- `WbAdapterTunnel`: rounded tunnel/portal with a cable passing through.
- `WbAdapterVirtual`: overlapping device/window tiles for virtual adapters.
- `WbEntityPeak`: rising peak line over a baseline with a top node.
